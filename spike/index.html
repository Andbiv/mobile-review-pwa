Mobile Review — Transport Spike

Validates that an iOS Safari PWA can read pack zips from OneDrive and write verdict files back. If this works, we proceed to build the real PWA. If not, we redesign the transport before sinking a week into the exporter.

Setup (do this once on the iPad)
  1. Open this page in Safari on the iPad.
  2. Tap the share button → Add to Home Screen → "ReviewSpike". This installs it as a PWA.
  3. Open ReviewSpike from the home screen (not Safari).
  4. Tap "Generate test pack" below → it'll download a small zip. When the Save sheet appears, save to OneDrive – BCI / MobileReview / test_pack.zip (create the folder if needed).
  5. Wait ~30 seconds for OneDrive to sync. Verify on the desktop that the file is there.

Test 1 — Read a zip from OneDrive

Pick the test pack you just saved (or any zip from OneDrive). Verify the PWA can read its contents through the iOS Files picker.

Test 3 — Read a pinned zip with Airplane Mode ON

The make-or-break test. Steps:

  1. In the OneDrive iOS app, find test_pack.zip. Tap "⋯" → "Make available offline". Wait for the download icon to confirm it's pinned.
  2. Enable Airplane Mode in iOS Control Center.
  3. Tap below to pick the same zip. It should read fine even with no network.
